Как расположить контент дашборда?

Выбери вариант, который нравится больше

A — Шире, та же структура

Убираем ограничение ширины, контент растягивается на весь экран. Карточки становятся крупнее, но структура остаётся прежней.

✓
↔ Контент растягивается на всю ширину экрана

B — Две большие колонки

Левая колонка: Hero + Курсы. Правая колонка: Путь развития + Достижения. Всё видно сразу, ничего не надо скроллить.

Лево: Hero + Курсы · Право: Путь развития + Достижения

C — Hero широкий, снизу три колонки

Hero и путь развития на всю ширину (выглядит солидно). Снизу три равных колонки: курсы, прогресс по уровням, достижения.

Hero + Путь полная ширина · снизу 3 колонки: Курсы · Уровень · Достижения
