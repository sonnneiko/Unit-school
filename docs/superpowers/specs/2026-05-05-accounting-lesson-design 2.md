# Spec: Урок «Аккаунтинг» — 3 темы

**Дата:** 2026-05-05  
**Статус:** Approved

## Контекст

Раздел «Аккаунтинг» в курсе Junior AM сейчас скрыт (`comingSoon: true`) и содержит 4 темы-заглушки. Нужно заменить на 3 реальных урока с новыми типами слайдов.

## Изменения в данных

### `src/data/courses.ts`

Секция `unitpay-accounting` — заменить 4 темы на 3, убрать `comingSoon`:

```ts
{
  id: 'unitpay-accounting',
  icon: '📊',
  title: 'Аккаунтинг',
  topics: [
    { id: 'intro',  icon: '🧑‍💼', title: 'Аккаунтинг',  lessonId: 'accounting-intro' },

    { id: 'chats',  icon: '💬',  title: 'Чаты',         lessonId: 'accounting-chats' },
  ],
}
```

### `src/data/lessons.ts`

Добавить 3 новых урока:

- `accounting-intro` — 1 слайд типа `welcome`
- `accounting-tools` — 1 слайд типа `tools`
- `accounting-chats` — 1 слайд типа `tgchats`

*Примечание: кириллические имена файлов в `src/assets/` корректно обрабатываются Vite через ESM import — файл `Аккаунтинг.png` уже импортируется в `WelcomeSlide.tsx` как `unitCat`, проблем с вебпаком нет.*

### 2. `tools` (новый тип) — урок «Инструменты»

**Лейаут:** два столбца на весь экран (`overflow: hidden`).

**Левая панель (340px):** вертикальный список из 4 карточек-инструментов. Каждая карточка — цветной градиентный блок с иконкой и названием. Активная подсвечивается белой рамкой.

**Правая панель:** название инструмента, описание, и блок для видео (`<iframe>` embed или placeholder с текстом «Видео скоро появится»).

**По умолчанию:** первый инструмент активен.

**Инструменты:**

| Название | Градиент | Иконка (Lucide) | Описание |
|---|---|---|---|
| Adminка UnitPay | `linear-gradient(135deg, #6BCB77, #4CAF50)` | `Settings` | Главный рабочий инструмент. Здесь всё: проекты, транзакции, настройки мерчантов, финансовые данные. |
| Omnidesk | `linear-gradient(135deg, #9B59B6, #6c5ce7)` | `MessageSquare` | Система для взаимодействия с мерчантами и плательщиками. Обращения, тикеты, история переписки. |
| Teamly | `linear-gradient(135deg, #F4A800, #e67e22)` | `LayoutDashboard` | Инструмент для взаимодействия с другими отделами. Задачи, документы, внутренние процессы. |
| Telegram | `linear-gradient(135deg, #2AABEE, #0984e3)` | `Send` | Взаимодействие с командой, партнёрами и поставщиками. |

**Новые типы в `src/types/index.ts`:**

```ts
export interface ToolItem {
  id: string

  gradient: string        // полная CSS строка: "linear-gradient(135deg, #AAA, #BBB)"

  description: string
  videoUrl?: string       // embed URL (YouTube/Loom), опционально
}

export interface ToolsContent {
  tools: ToolItem[]
}
```

**Компонент:** `src/components/slides/ToolsSlide.tsx`  
**Стили:** классы в `src/components/slides/slides.module.css`

### 3. `tgchats` (новый тип) — урок «Чаты»

**Лейаут:** два столбца на весь экран (`overflow: hidden`).

**Левая панель (340px, тёмная `#1c2733`):** имитация интерфейса Telegram.
- Заголовок «Чаты» + вкладки «Все» / «Триггеры»
- Вкладка «Все» — показывает все 7 чатов. Вкладка «Триггеры» — фильтрует: показывает только чаты с `cardId === 'triggers'` (3 чата).
- Список чатов с аватарками, названиями, превью последнего сообщения, бейджами непрочитанных
- Активный чат подсвечивается `#2b5278`

**Правая панель (белая):** 
- Заголовок «Рабочие чаты»
- Карточка описания выбранного чата (анимация fadeIn при смене)
- Внизу: примечание + кнопка «Скачать список чатов» (ведёт на PDF-файл)
